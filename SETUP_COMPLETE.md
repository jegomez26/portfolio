## 🎨 Customization Options

### Change Primary Color
Edit line 2 in `css/styles.css`:
```css
--primary-color: #667eea;  /* Change to your brand color */
```

### Add More Projects
Copy the project-card structure in index.html:
```html
<div class="project-card">
    <!-- Copy project structure from existing cards -->
</div>
```

### Add More Skills
Copy skill-category in index.html:
```html
<div class="skill-category">
    <!-- Copy skill category structure -->
</div>
```
